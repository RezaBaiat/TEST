RootDispatcher.js

Calling each method in this class will result to the related action to be executed in store of RootReducer and also grants possibility to call them statically without any need to import functions upon every usage providing a lot of benefits including cleaner code , easier future patches , IDE code completion and â€¦

import store from '../../configs/redux/Store';
import RootActions from '../actions/RootActions';

export default class RootDispatcher {

  static setNetworkState(isConnected : boolean) {
    store.dispatch(RootActions.networkStateChangeAction(isConnected));
  }

  static setData(newData : string) {
    store.dispatch(RootActions.setDataAction(newData));
  }

  static runSagaTest(url : string) {
    store.dispatch(RootActions.testSagaAction(url));
  }

}
